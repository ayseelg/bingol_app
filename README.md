# Bingöl App (Android)

Bu proje, Bingöl ile ilgili çeşitli işlemleri veya içerikleri sunan temel bir Android uygulamasıdır. İçerisinde internet, kamera ve depolama erişimleri gibi çeşitli özellikler için izinler tanımlanmıştır.

## 🚀 Özellikler
* `HomeActivity` ve `MainActivity` ile ayrılmış temel navigasyon yapısı
* İnternet, kamera ve depolama erişim izinleri konfigürasyonu
* Temel Android projesi altyapısı (Gradle)
* Arka plan güvenlik ayarları ve tema uygulamaları (`Theme.Bingol`)

## 🛠️ Gereksinimler
* Android Studio (En güncel sürümü önerilir)
* Minimum Android API seviyesi: Gradle konfigürasyonuna göre test edilmelidir.

## 💻 Kurulum ve Çalıştırma
1. Projeyi bilgisayarınıza klonlayın veya zip olarak indirip çıkartın.
2. Android Studio'yu açın ve **"Open"** (Aç) seçeneği ile bu projenin bulunduğu dizini seçin.
3. Android Studio'nun Gradle bağımlılıklarını indirmesini ve projeyi yapılandırmasını bekleyin (`Sync`).
4. Fiziksel bir Android cihaz bağlayın veya Android Studio içinden bir Emülatör (AVD) başlatın.
5. Üst menüden yeşil "Play" (Run) tuşuna basarak projeyi derleyip çalıştırın.
